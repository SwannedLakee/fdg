Меню Voice

Нажмите на ссылку Voice на странице текста, чтобы открыть панель управления звуком.

1. Встроенный плеер

Использует синтез речи браузера.

  • Управление: Кнопки Play/Pause, предыдущее и следующее предложения.
  • Навигация: Нажмите паузу. Кликните на любое предложение пали, чтобы оно выделилось цветом. Нажмите Play, чтобы проигрывать с него.
  • Настройки: Выбор режима (Пали / Перевод / Пали+Перевод / Перевод + Пали) и скорости чтения.

Для синтеза речи близкой к пали активируйте один из голосовых модулей на своём устройстве: Sanskrit (India), Hindi (India), Nepali или Indonesian. Отдельного TTS-модуля для пали не существует. Если ни один из них не установлен, будет использован английский как запасной вариант.

Android: Настройки → Язык и ввод → Синтез речи → Установка голосовых данных.
iOS: Настройки → Универсальный доступ → Озвучивание содержимого → Голоса.
ПК / macOS: Установите системный голос через настройки операционной системы (см. инструкции в интернете).

2. Дополнительные ссылки

В нижней части панели Voice доступны следующие опции:

  • TTS: Открывает специальную страницу для использования внешних TTS-приложений.

    Примеры приложений:

    • Android: Play Market, apk local
    • iOS: Voice Aloud Reader
    • Chrome: Read Aloud
  • VSC: Переход на сайт Voice.SC (SuttaCentral Voice) для прослушивания текста.
  • File: (Если доступен) Прямая ссылка на аудиофайл. Это может быть сохранённый TTS-файл или запись чтения на пали.
  • ?: Ссылка на эту страницу помощи.
3. Google API Key (необязательно)

Позволяет использовать облачный синтез речи Google (Google Cloud Text-to-Speech). Хорошие голоса и большие бесплатные лимиты.
Как получить ключ: создайте API-ключ в Google Cloud Console и включите Text-to-Speech API.
Подробную инструкцию можно найти в интернете по запросу: google tts cloud api key

Dhamma was originally an oral teaching.

"Pañcime, bhikkhave, ādīnavā āyatakena gītassarena dhammaṁ bhaṇantassa."
"There are, monks, these five dangers in reciting the Dhamma with a drawn-out singing voice."

an5.209


Voice Menu

Click the Voice link on a text page to open the audio control panel.

1. Built-in Player

Uses your browser's speech synthesis.

  • Controls: Play/Pause buttons, sentence navigation.
  • Navigation: Press Pause. Click on any Pali sentence to highlight it. Press Play to start playing from there.
  • Settings: Select reading mode (Pali / Translation / Pali+Translation / Translation+Pali) and speed.

To listen to Pali, enable one of the following voice modules on your device: Sanskrit (India), Hindi (India), Nepali or Indonesian. There is no dedicated TTS engine for Pali. If none of these are installed, English will be used as a fallback.

Android: Settings → Language & Input → Text-to-Speech → Install voice data.
iOS: Settings → Accessibility → Spoken Content → Voices.
PC / macOS: Install a system voice via your OS settings (see instructions online).

2. Extra Links (Footer)

At the bottom of the Voice panel, you will find:

  • TTS: Opens a dedicated page for external TTS apps.

    Example apps:

    • Android: Play Market, apk local
    • iOS: Voice Aloud Reader
    • Chrome: Read Aloud
  • VSC: Link to Voice.SC (SuttaCentral Voice).
  • File: (If available) Direct link to an audio file. This may be a saved TTS file or a Pali reading recording.
  • ?: Link to this help page.
3. Google API Key (optional)

Enables Google Cloud Text-to-Speech. Provides higher-quality voices and generous free limits.
How to get a key: create an API key in Google Cloud Console and enable Text-to-Speech API.
Search online for: google tts cloud api key
