Слушание Дхаммы

Дхамма изначально была устным учением.

"Pañcime, bhikkhave, ādīnavā āyatakena gītassarena dhammaṁ bhaṇantassa."
"Вот, монахи, пять опасностей для того, кто декламирует Дхамму протяженным песенным голосом."

an5.209

На Dhamma.gift предусмотрено несколько способов прослушать Сутты и другие тексты на пали или языке перевода:

1. Встроенный аудиоплеер

На страницах Сутт и Винаи при клике на ссылку "Voice", если есть сохранённый файл, откроется аудиоплеер с записью Пали текста. В основном это файлы TTS (text-to-speech) из проекта voice.suttacentral.net. Это встроенный плеер вашего браузера. Также обычно такие плееры позволяют сохранить аудио на устройство.

Если файла нет, то будут только дополнительные ссылки.

2. Копирование текста для TTS-приложений

Некоторые TTS приложения проигрывают текст скопированный в буфер обмена. Используйте кнопки "Pali Copy" или "Trn Copy", чтобы скопировать нужный текст для воспроизведения.

Другие приложения проигрывают текст со страницы. Нажмите "Pali Open" или "Trn Open", чтобы открыть страницу только с нужным текстом, который можно будет воспроизвести. Если данная опция не работает, пожалуйста, попробуйте другой браузер. Должно работать в Edge, Chrome, Firefox, Opera, Safari.

Для прослушивания пали можно использовать санскритские TTS-модули. Будут небольшие различия в чтении, но у вас будет очень широкий выбор приложений, сайтов, расширений, голосов и настроек. Если для модуля TTS потребуется текст в деванагари, эту опцию можно включить в настройках интерфейса чтения: "Шестерёнка" → Система письма → Devanagari → Применить.

Пример приложений для Android, iOS и расширений для Chrome:

  • Android: Text to Speech
  • iOS: Voice Aloud Reader
  • Или встроенные функции TTS вашего устройства
  • Расширение Chrome: Read Aloud
3. Voice.SC

Voice.SC — TTS-сайт с возможностью прослушивания пали и второго языка "построчно".

Listening to Dhamma

Dhamma was originally an oral teaching.

"Pañcime, bhikkhave, ādīnavā āyatakena gītassarena dhammaṁ bhaṇantassa." "There are, monks, these five dangers in reciting the Dhamma with a drawn-out singing voice."

an5.209

There are several ways on Dhamma.Gift to listen to suttas and other texts in Pali or in translation:

1. Built-in audio player

On Sutta and Vinaya pages, clicking the "Voice" link will open an audio player with a audio of the Pali text being read, if a saved file is available. These are mostly saved TTS (text-to-speech) files from the voice.suttacentral.net project. It wiil be your browser's built-in player will appear. Such players typically allow you to save the audio file on your device.

If no file is available, the "Voice" link will have only extra links.

2. Copying text for TTS applications

Some TTS apps use Copied text in Clipboard to play it. Use the "Pali Copy" or "Trn Copy" (copy translation) buttons to quickly copy text for these apps.

Some TTS apps read text from pages directly. Use the "Pali Open" or "Trn Open" buttons to get single language page. Is this option doesn't work, please try another browser or browser verion. Should work in Edge, Chrome, Firefox, Opera, Safari.

For listening to Pali, you can use Sanskrit TTS modules with minor differences in pronunciation, giving you a wide selection of applications, websites, extensions, voices, and settings. If the TTS module requires text in Devanagari, you can enable this option in the reading interface settings: "Gear icon" → Writing System → Devanagari → Apply.

Example apps for Android and iOS:

  • Android: Text to Speech
  • iOS: Voice Aloud Reader
  • Or your device's built-in TTS features
  • Chrome Extention: Read Aloud
3. Voice.SC

Voice.SC is a TTS website with the ability to listen to Pali and a second language "line by line".
