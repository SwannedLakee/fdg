Настройки словаря по клику

Чтобы выбрать стиль словаря, который лучше всего соответствует вашим потребностям:

  1. Откройте Dhamma.gift Read.
  2. Нажмите на значок шестерёнки ⚙️ (Настройки).
  3. Нажмите на выпадающий список "Словарь по клику" и выберите нужный вариант.
  4. Нажмите "Применить" для сохранения настроек.

DPD Онлайн Рус: Полная версия словаря DPD с расширенными грамматическими деталями, информацией о корнях, примерами и другим. С возможностью ввода текста для дальнейшего исследования, историей поиска и настройками.

DPD Онлайн мини Рус: Компактная версия полного словаря DPD с основными деталями.

DPD Встроенный: Пали-Английский. Облегчённая, но мощная встроенная версия Digital Pali Dictionary (DPD), предоставляющая основные функции: значения слов, разбор сложных слов и базовую грамматическую информацию.

DPD Онлайн: Полный словарь DPD с английскими переводами и объяснениями.

DPD Онлайн мини: Компактная версия английского словаря DPD.

Только поиск: Эта опция позволяет мгновенно начать поиск кликнутого слова на Dhamma.gift.

DictTango Android: Прямое открытие приложения DictTango на Android, позволяющее мгновенный поиск в любых установленных словарях, включая DPD, без копирования, вставки или использования контекстного меню.
