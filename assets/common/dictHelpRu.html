Настройки словаря по клику

Чтобы выбрать стиль словаря, который лучше всего соответствует вашим потребностям:

  1. Откройте Dhamma.gift Read.
  2. Нажмите на значок шестерёнки ⚙️ (Настройки).
  3. Нажмите на выпадающий список "Словарь по клику" и выберите нужный вариант.
  4. Нажмите "Применить" для сохранения настроек.

DPD Встроенный: Пали-русский. Облегчённая, но мощная встроенная версия Digital Pali Dictionary (DPD), предоставляющая основные функции: значения слов, разбор сложных слов и базовую грамматическую информацию. Жирным выделены проверенные слова, без выделения — машинный перевод и записи, оставленные без перевода из английской версии DPD.

DPD Встроенный Англ: облегчённый Пали-английский DPD

DPD Онлайн: Полная русская версия словаря DPD с расширенными грамматическими деталями, информацией о корнях, примерами и другим. С возможностью ввода текста для дальнейшего исследования, историей поиска и настройками.

DPD Онлайн Англ: Полный словарь DPD с английскими переводами и объяснениями.

DPD Онлайн мини: Компактная русская версия полного словаря DPD с основными вкладками, без возможности перехода по словам внутри словаря, без поля ввода текста, без настроек и истории поиска.

DPD Онлайн мини Англ: Компактная версия английского словаря DPD.

Только поиск: Эта опция позволяет мгновенно начать поиск кликнутого слова на Dhamma.gift.

DictTango Android: Прямое открытие приложения DictTango Android, позволяющее мгновенный поиск в любых установленных словарях, включая DPD, без копирования, вставки или использования контекстного меню + Кнопка Лупы при двойном нажатии запускает поиск слова на Dhamma.gift.



Mdict IOS: Также как предыдущий пункт, но для Mdict IOS. Прямое открытие приложения Mdict IOS соответственно.



Онлайн-версии зависят от интернет-соединения, поэтому могут работать медленнее, чем встроенная, но дополнительные функции того стоят.
